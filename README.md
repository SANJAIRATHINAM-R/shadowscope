## 🚀 Overview

ShadowScope is a lightweight CLI-based cybersecurity tool designed to detect and analyze shadow IT activity.

It simulates real-world security monitoring by scanning domains, analyzing risk levels, and generating alerts.

---

## 🎯 Purpose

This project helps:

- 🧠 Learn real-world cybersecurity workflows  
- 🛡️ Simulate enterprise-level threat detection  
- ⚔️ Build an attacker mindset  
- 🎯 Practice ethical hacking concepts  

---

## ⚡ Features

- 🔍 URL Risk Analysis  
- 🚨 High Risk Detection  
- 📡 Activity Logging  
- ⚙️ Backend API (Node.js)  
- 🧾 CLI Tool (Bash)  
- 🧠 Smart Detection Logic
